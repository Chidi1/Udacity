How to lazy load images in React

Example Code

I will be using react-lazyload library (https://github.com/twobin/react-lazyload) to lazy load the images.

Install the dependencies using the following command.

npm install --save react-lazyload

Once you have installed the library. Use the following syntax to lazy load the images.

<LazyLoad width={<width>} height={<height>}>
    <img src={<image-url>}/>
</LazyLoad>

Example:

<LazyLoad width={232} height={300}>
    <img src="https://in.bmscdn.com/events/moviecard/ET00046165.jpg"/>
</LazyLoad>
